# Get Flow

Retrieve comprehensive information about a specific workflow, including its complete definition, metadata, execution statistics, and current validation status.

## Endpoint

<CodeGroup>
```http HTTP
GET /flows/{id}
```

```bash cURL
curl -X GET "https://tolstoy.getpullse.com/flows/flow_abc123def456" \
  -H "x-org-id: org_1234567890abcdef" \
  -H "x-user-id: user_abcdef1234567890"
```
</CodeGroup>

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | **Required.** The unique identifier of the flow to retrieve |

## Response Fields

### Core Information

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique flow identifier |
| `name` | string | Human-readable flow name |
| `description` | string | Detailed flow description |
| `version` | integer | Current version number |
| `status` | string | Flow status (`active`, `inactive`, `draft`) |
| `tags` | array | Categorization tags |

### Flow Definition

| Field | Type | Description |
|-------|------|-------------|
| `definition.inputs` | array | Input parameter definitions with validation |
| `definition.steps` | array | Workflow steps with actions and logic |
| `definition.outputs` | array | Output parameter mappings |
| `definition.settings` | object | Execution configuration and policies |

### Metadata

| Field | Type | Description |
|-------|------|-------------|
| `created_at` | string | ISO 8601 creation timestamp |
| `updated_at` | string | ISO 8601 last modification timestamp |
| `created_by` | object | User who created the flow |
| `updated_by` | object | User who last modified the flow |

### Validation

| Field | Type | Description |
|-------|------|-------------|
| `validation.valid` | boolean | Whether the flow definition is valid |
| `validation.warnings` | array | Non-blocking validation warnings |
| `validation.errors` | array | Blocking validation errors |

### Statistics

| Field | Type | Description |
|-------|------|-------------|
| `stats.total_executions` | number | Total number of executions |
| `stats.successful_executions` | number | Number of successful executions |
| `stats.failed_executions` | number | Number of failed executions |
| `stats.success_rate` | number | Success rate (0.0 to 1.0) |
| `stats.avg_duration_ms` | number | Average execution duration in milliseconds |
| `stats.last_execution_at` | string | Timestamp of most recent execution |

## Request Examples

<CodeGroup>
```bash cURL - Basic Request
curl -X GET "https://tolstoy.getpullse.com/flows/flow_abc123def456" \
```

```typescript TypeScript/Node.js
import { TolstoyClient } from '@joosuhail/tolstoy-sdk';

const client = new TolstoyClient({
  orgId: 'org_1234567890abcdef',
  userId: 'user_abcdef1234567890'
});

try {
  const flow = await client.flows.get('flow_abc123def456');
  
  console.log('Flow details:', {
    name: flow.name,
    version: flow.version,
    status: flow.status,
    successRate: (flow.stats.success_rate * 100).toFixed(1) + '%',
    totalExecutions: flow.stats.total_executions
  });
  
  // Check if flow is ready for production
  if (flow.validation.valid && flow.validation.errors.length === 0) {
    console.log('Flow is production ready!');
  } else {
    console.log('Flow has validation issues:', flow.validation.errors);
  }
  
} catch (error) {
  console.error('Failed to retrieve flow:', error);
}
```

```javascript JavaScript/Fetch
async function getFlowDetails(flowId) {
  try {
    const response = await fetch(`https://tolstoy.getpullse.com/flows/${flowId}`, {
      method: 'GET',
      headers: {
        'x-org-id': 'org_1234567890abcdef',
        'x-user-id': 'user_abcdef1234567890',
        'Content-Type': 'application/json'
      }
    });
    
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    
    const result = await response.json();
    const flow = result.data;
    
    return {
      id: flow.id,
      name: flow.name,
      status: flow.status,
      version: flow.version,
      stepCount: flow.definition.steps.length,
      executionStats: {
        total: flow.stats.total_executions,
        successRate: flow.stats.success_rate,
        avgDuration: flow.stats.avg_duration_ms
      },
      isValid: flow.validation.valid,
      warnings: flow.validation.warnings,
      errors: flow.validation.errors
    };
    
  } catch (error) {
    console.error('Error fetching flow:', error);
    throw error;
  }
}

// Usage
getFlowDetails('flow_abc123def456')
  .then(flow => {
    console.log('Flow loaded:', flow.name);
    console.log(`Performance: ${(flow.executionStats.successRate * 100).toFixed(1)}% success rate`);
  })
  .catch(error => console.error('Failed to load flow:', error));
```

## Use Cases

### Flow Inspection and Debugging

```typescript
async function inspectFlow(flowId: string) {
  const flow = await client.flows.get(flowId);
  
  console.log(`=== Flow Analysis: ${flow.name} ===`);
  console.log(`Version: ${flow.version}`);
  console.log(`Status: ${flow.status}`);
  console.log(`Steps: ${flow.definition.steps.length}`);
  
  // Analyze validation issues
  if (!flow.validation.valid) {
    console.log('\n❌ VALIDATION ERRORS:');
    flow.validation.errors.forEach((error, i) => {
      console.log(`  ${i + 1}. ${error}`);
    });
  }
  
  if (flow.validation.warnings.length > 0) {
    console.log('\n⚠️  WARNINGS:');
    flow.validation.warnings.forEach((warning, i) => {
      console.log(`  ${i + 1}. ${warning}`);
    });
  }
  
  // Performance analysis
  const successRate = (flow.stats.success_rate * 100).toFixed(1);
  const avgDuration = flow.stats.avg_duration_ms / 1000;
  
  console.log(`\n📊 PERFORMANCE:`);
  console.log(`  Success Rate: ${successRate}%`);
  console.log(`  Avg Duration: ${avgDuration.toFixed(1)}s`);
  console.log(`  Total Executions: ${flow.stats.total_executions.toLocaleString()}`);
  
  // Step analysis
  console.log(`\n🔍 STEP BREAKDOWN:`);
  flow.definition.steps.forEach((step, i) => {
    console.log(`  ${i + 1}. ${step.name} (${step.action})`);
    if (step.executeIf) {
      console.log(`     Condition: ${step.executeIf}`);
    }
  });
  
  return {
    needsAttention: !flow.validation.valid || flow.stats.success_rate < 0.9,
    recommendations: generateRecommendations(flow)
  };
}

function generateRecommendations(flow: any): string[] {
  const recommendations = [];
  
  if (flow.stats.success_rate < 0.9) {
    recommendations.push('Consider reviewing failed executions to improve success rate');
  }
  
  if (flow.stats.avg_duration_ms > 30000) {
    recommendations.push('Flow execution time is high - review step efficiency');
  }
  
  if (flow.validation.warnings.length > 2) {
    recommendations.push('Address validation warnings to improve flow reliability');
  }
  
  if (flow.definition.steps.length > 10) {
    recommendations.push('Consider breaking complex flow into smaller, focused workflows');
  }
  
  return recommendations;
}
```

### Flow Comparison

```typescript
async function compareFlows(flowId1: string, flowId2: string) {
  const [flow1, flow2] = await Promise.all([
    client.flows.get(flowId1),
    client.flows.get(flowId2)
  ]);
  
  return {
    performance: {
      flow1: {
        name: flow1.name,
        successRate: flow1.stats.success_rate,
        avgDuration: flow1.stats.avg_duration_ms,
        executions: flow1.stats.total_executions
      },
      flow2: {
        name: flow2.name,
        successRate: flow2.stats.success_rate,
        avgDuration: flow2.stats.avg_duration_ms,
        executions: flow2.stats.total_executions
      },
      winner: flow1.stats.success_rate > flow2.stats.success_rate ? flow1.name : flow2.name
    },
    complexity: {
      flow1: {
        steps: flow1.definition.steps.length,
        inputs: flow1.definition.inputs.length,
        outputs: flow1.definition.outputs?.length || 0
      },
      flow2: {
        steps: flow2.definition.steps.length,
        inputs: flow2.definition.inputs.length,
        outputs: flow2.definition.outputs?.length || 0
      }
    },
    validation: {
      flow1Valid: flow1.validation.valid,
      flow2Valid: flow2.validation.valid,
      flow1Warnings: flow1.validation.warnings.length,
      flow2Warnings: flow2.validation.warnings.length
    }
  };
}
```
