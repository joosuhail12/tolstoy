# Get Action

<Badge variant="success">GET</Badge> `/actions/{action_id}`

Retrieve detailed information about a specific action by its ID.

## Request

### **HTTP Request**

```http
GET https://api.tolstoy.dev/actions/{action_id}
```

### **Path Parameters**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `action_id` | string | Yes | The unique identifier of the action |

### **Query Parameters**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `include_statistics` | boolean | true | Include execution statistics |
| `include_versions` | boolean | false | Include version history |

## Examples

### **Basic Request**

<CodeGroup>

```bash cURL
curl -X GET 'https://api.tolstoy.dev/actions/act_abc123xyz456' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json'
```

```javascript JavaScript
const actionId = 'act_abc123xyz456';

const response = await fetch(`https://api.tolstoy.dev/actions/${actionId}`, {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log('Action details:', data.action);
console.log('Success rate:', `${(data.action.statistics.success_rate * 100).toFixed(2)}%`);
```

```python Python
import requests

action_id = 'act_abc123xyz456'

response = requests.get(
    f'https://api.tolstoy.dev/actions/{action_id}',
    headers={
        'Authorization': f'Bearer {api_key}',
        'Content-Type': 'application/json'
    }
)

if response.status_code == 200:
    action = response.json()['action']
    print(f"Action: {action['display_name']}")
    print(f"Type: {action['type']}")
    print(f"Status: {action['status']}")
    print(f"Success rate: {action['statistics']['success_rate']:.2%}")
```

</CodeGroup>

### **Request with Version History**

<CodeGroup>

```bash Include Versions
curl -X GET 'https://api.tolstoy.dev/actions/act_abc123xyz456?include_versions=true' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json'
```

```javascript JavaScript with Options
const actionId = 'act_abc123xyz456';
const params = new URLSearchParams({
  include_statistics: 'true',
  include_versions: 'true'
});

const response = await fetch(
  `https://api.tolstoy.dev/actions/${actionId}?${params}`,
  {
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();

// Display version history
data.action.versions.forEach(version => {
  console.log(`Version ${version.version} - ${version.released_at}`);
  version.changes.forEach(change => {
    console.log(`  - ${change}`);
  });
});
```

</CodeGroup>
