# List Actions

<Badge variant="success">GET</Badge> `/actions`

Retrieve a paginated list of actions available in your organization. Actions are individual workflow steps that can be executed independently or combined into flows.

## Request

### **HTTP Request**

```http
GET https://api.tolstoy.dev/actions
```

### **Query Parameters**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | 20 | Number of actions per page (1-100) |
| `cursor` | string | - | Pagination cursor for next page |
| `sort` | string | `created_at` | Sort field (`name`, `created_at`, `updated_at`, `last_executed`) |
| `order` | string | `desc` | Sort order (`asc` or `desc`) |
| `search` | string | - | Search actions by name or description |
| `type` | string | - | Filter by action type (`http`, `database`, `script`, `integration`) |
| `status` | string | - | Filter by status (`active`, `draft`, `deprecated`) |
| `tag` | string | - | Filter by tag |
| `created_after` | string | - | Filter by creation date (ISO 8601) |
| `created_before` | string | - | Filter by creation date (ISO 8601) |

### **Action Object**

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique action identifier |
| `name` | string | Action name (used in API and flows) |
| `display_name` | string | Human-readable action name |
| `description` | string | Detailed action description |
| `type` | string | Action type (`http`, `database`, `script`, `integration`) |
| `status` | string | Action status (`active`, `draft`, `deprecated`) |
| `version` | string | Semantic version number |
| `category` | string | Action category for organization |
| `tags` | array | Tags for filtering and organization |
| `configuration` | object | Input/output configuration and settings |
| `permissions` | object | Required permissions and rate limits |
| `statistics` | object | Execution statistics |
| `created_by` | string | User ID who created the action |
| `created_at` | string | ISO 8601 timestamp of creation |
| `updated_at` | string | ISO 8601 timestamp of last update |

## Examples

### **Basic Request**

<CodeGroup>

```bash cURL
curl -X GET 'https://api.tolstoy.dev/actions' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json'
```

```javascript JavaScript
const response = await fetch('https://api.tolstoy.dev/actions', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(`Found ${data.actions.length} actions`);

data.actions.forEach(action => {
  console.log(`- ${action.display_name} (${action.id})`);
  console.log(`  Type: ${action.type}, Status: ${action.status}`);
  console.log(`  Executions: ${action.statistics.total_executions}`);
});
```

```python Python
import requests

response = requests.get(
    'https://api.tolstoy.dev/actions',
    headers={
        'Authorization': f'Bearer {api_key}',
        'Content-Type': 'application/json'
    }
)

data = response.json()
print(f"Found {len(data['actions'])} actions")

for action in data['actions']:
    print(f"- {action['display_name']} ({action['id']})")
    print(f"  Type: {action['type']}, Status: {action['status']}")
    print(f"  Executions: {action['statistics']['total_executions']}")
```

</CodeGroup>

### **Pagination Example**

<CodeGroup>

```javascript Paginate Through All Actions
async function getAllActions(apiKey) {
  const actions = [];
  let cursor = null;
  
  while (true) {
    const params = new URLSearchParams({ limit: '100' });
    if (cursor) {
      params.set('cursor', cursor);
    }
    
    const response = await fetch(`https://api.tolstoy.dev/actions?${params}`, {
      headers: {
        'Authorization': `Bearer ${apiKey}`,
        'Content-Type': 'application/json'
      }
    });
    
    const data = await response.json();
    actions.push(...data.actions);
    
    if (!data.pagination.has_more) {
      break;
    }
    
    cursor = data.pagination.next_cursor;
  }
  
  return actions;
}

// Usage
const allActions = await getAllActions(apiKey);
console.log(`Total actions: ${allActions.length}`);
```

</CodeGroup>
