# List Tools

<Badge variant="success">GET</Badge> `/tools`

Retrieve a paginated list of tools available in your organization. Tools are external integrations and services that can be used within actions and flows.

## Request

### **HTTP Request**

```http
GET https://api.tolstoy.dev/tools
```

### **Query Parameters**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | 20 | Number of tools per page (1-100) |
| `cursor` | string | - | Pagination cursor for next page |
| `sort` | string | `created_at` | Sort field (`name`, `created_at`, `updated_at`, `last_used`) |
| `order` | string | `desc` | Sort order (`asc` or `desc`) |
| `search` | string | - | Search tools by name or description |
| `category` | string | - | Filter by category (`database`, `communication`, `storage`, `analytics`) |
| `status` | string | - | Filter by status (`active`, `inactive`, `error`) |
| `provider` | string | - | Filter by provider (`salesforce`, `slack`, `postgres`, `s3`) |

### **Tool Object**

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique tool identifier |
| `name` | string | Tool name (used in actions) |
| `display_name` | string | Human-readable tool name |
| `description` | string | Detailed tool description |
| `provider` | string | Tool provider/service type |
| `category` | string | Tool category |
| `status` | string | Tool status (`active`, `inactive`, `error`) |
| `version` | string | Tool/service version |
| `connection` | object | Connection configuration |
| `credentials` | object | Authentication configuration |
| `capabilities` | array | Available operations |
| `health` | object | Current health status |
| `usage` | object | Usage statistics |
| `tags` | array | Tags for organization |
| `created_by` | string | User who created the tool |
| `created_at` | string | ISO 8601 timestamp of creation |
| `updated_at` | string | ISO 8601 timestamp of last update |

### **Filter by Category**

<CodeGroup>

```bash Filter Database Tools
curl -X GET 'https://api.tolstoy.dev/tools?category=database&status=active' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json'
```

```javascript JavaScript with Filters
const params = new URLSearchParams({
  category: 'communication',
  status: 'active',
  sort: 'last_used',
  order: 'desc'
});

const response = await fetch(`https://api.tolstoy.dev/tools?${params}`, {
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  }
});

const data = await response.json();

// Find tools with high usage
const highUsageTools = data.tools.filter(tool => {
  if (tool.category === 'database') {
    return tool.usage.queries_last_24h > 1000;
  } else if (tool.category === 'communication') {
    return tool.usage.messages_last_24h > 50;
  }
  return false;
});

console.log(`High usage tools: ${highUsageTools.length}`);
```

### **Tool Health Monitor**

<CodeGroup>

```javascript Monitor Tool Health
class ToolHealthMonitor {
  constructor(apiKey, alertThresholds = {}) {
    this.apiKey = apiKey;
    this.thresholds = {
      responseTimeMs: 1000,
      errorCount24h: 5,
      connectionThreshold: 0.8,
      ...alertThresholds
    };
  }
  
  async checkAllTools() {
    const response = await fetch('https://api.tolstoy.dev/tools', {
      headers: {
        'Authorization': `Bearer ${this.apiKey}`,
        'Content-Type': 'application/json'
      }
    });
    
    const { tools } = await response.json();
    const issues = [];
    
    tools.forEach(tool => {
      const health = tool.health;
      const usage = tool.usage;
      
      // Check response time
      if (health.response_time_ms > this.thresholds.responseTimeMs) {
        issues.push({
          tool: tool.display_name,
          type: 'performance',
          message: `High response time: ${health.response_time_ms}ms`,
          severity: 'warning'
        });
      }
      
      // Check error count
      if (health.error_count_24h > this.thresholds.errorCount24h) {
        issues.push({
          tool: tool.display_name,
          type: 'reliability',
          message: `High error count: ${health.error_count_24h} errors in 24h`,
          severity: 'error'
        });
      }
      
      // Database-specific checks
      if (tool.category === 'database') {
        const connectionUsage = health.connection_count / tool.connection.pool_size;
        if (connectionUsage > this.thresholds.connectionThreshold) {
          issues.push({
            tool: tool.display_name,
            type: 'capacity',
            message: `High connection usage: ${Math.round(connectionUsage * 100)}%`,
            severity: 'warning'
          });
        }
      }
      
      // Storage-specific checks
      if (tool.category === 'storage' && health.storage_used_gb > 1000) {
        issues.push({
          tool: tool.display_name,
          type: 'capacity',
          message: `High storage usage: ${health.storage_used_gb}GB`,
          severity: 'info'
        });
      }
    });
    
    return { tools, issues };
  }
  
  async generateHealthReport() {
    const { tools, issues } = await this.checkAllTools();
    
    console.log('Tool Health Report');
    console.log('==================');
    
    const byCategory = tools.reduce((acc, tool) => {
      acc[tool.category] = acc[tool.category] || [];
      acc[tool.category].push(tool);
      return acc;
    }, {});
    
    Object.entries(byCategory).forEach(([category, categoryTools]) => {
      console.log(`\n${category.toUpperCase()} (${categoryTools.length} tools):`);
      
      categoryTools.forEach(tool => {
        const status = tool.health.status === 'healthy' ? '‚úÖ' : '‚ùå';
        console.log(`  ${status} ${tool.display_name}`);
        console.log(`     Response: ${tool.health.response_time_ms}ms`);
        console.log(`     Errors: ${tool.health.error_count_24h}/24h`);
      });
    });
    
    if (issues.length > 0) {
      console.log('\nISSUES DETECTED:');
      issues.forEach(issue => {
        const icon = issue.severity === 'error' ? 'üî¥' : 
                    issue.severity === 'warning' ? 'üü°' : '‚ÑπÔ∏è';
        console.log(`${icon} ${issue.tool}: ${issue.message}`);
      });
    } else {
      console.log('\n‚úÖ All tools are healthy!');
    }
    
    return { tools, issues };
  }
}

// Usage
const monitor = new ToolHealthMonitor(apiKey);
await monitor.generateHealthReport();
```
