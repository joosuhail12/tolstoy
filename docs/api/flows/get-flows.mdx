# List Flows

Retrieve a paginated list of flows for your organization with optional filtering, sorting, and search capabilities.

<RequestExample>

```bash cURL
curl -X GET "https://tolstoy.getpullse.com/flows?limit=20&search=onboarding&status=active" \
  -H "x-org-id: org_1234567890abcdef" \
  -H "x-user-id: user_abcdef1234567890" \
  -H "Content-Type: application/json"
```

```typescript TypeScript
const response = await fetch('https://tolstoy.getpullse.com/flows?limit=20&search=onboarding', {
  method: 'GET',
  headers: {
    'x-org-id': 'org_1234567890abcdef',
    'x-user-id': 'user_abcdef1234567890',
    'Content-Type': 'application/json'
  }
});

const result = await response.json();
console.log('Flows:', result.data);
```

```python Python
import requests

headers = {
    'x-org-id': 'org_1234567890abcdef',
    'x-user-id': 'user_abcdef1234567890',
    'Content-Type': 'application/json'
}

params = {
    'limit': 20,
    'search': 'onboarding',
    'status': 'active'
}

response = requests.get(
    'https://tolstoy.getpullse.com/flows',
    headers=headers,
    params=params
)

flows = response.json()
print(f"Found {len(flows['data'])} flows")
```

```javascript Node.js

```

</RequestExample>

## Query Parameters

<ParamField query="limit" type="integer" default="20">
Number of flows to return per page. Maximum value is 100.
</ParamField>

<ParamField query="cursor" type="string">
Pagination cursor for retrieving the next page of results. Use the cursor from the previous response's pagination object.
</ParamField>

<ParamField query="search" type="string">
Search flows by name or description. Supports partial matching and is case-insensitive.
</ParamField>

<ParamField query="status" type="string">
Filter flows by status. Available values:
- `draft` - Flows in development
- `active` - Published and executable flows  
- `inactive` - Disabled flows
- `archived` - Archived flows
</ParamField>

<ParamField query="tags" type="string">
Filter flows by tags. Comma-separated list (e.g., `automation,customer-service`).
</ParamField>

<ParamField query="created_by" type="string">
Filter flows by creator user ID (e.g., `user_abcdef1234567890`).
</ParamField>

<ParamField query="sort_by" type="string" default="updated_at">
Sort field. Available values:
- `name` - Sort by flow name
- `created_at` - Sort by creation date
- `updated_at` - Sort by last modification date
- `status` - Sort by flow status
</ParamField>

<ParamField query="sort_order" type="string" default="desc">
Sort direction: `asc` (ascending) or `desc` (descending)
</ParamField>

## Response Fields

### Flow Object

<ResponseField name="id" type="string">
Unique flow identifier (format: `flow_` + 15 alphanumeric characters)
</ResponseField>

<ResponseField name="name" type="string">
Human-readable flow name
</ResponseField>

<ResponseField name="description" type="string | null">
Optional flow description
</ResponseField>

<ResponseField name="status" type="string">
Current flow status: `draft`, `active`, `inactive`, or `archived`
</ResponseField>

<ResponseField name="version" type="integer">
Current flow version number (increments on each update)
</ResponseField>

<ResponseField name="tags" type="string[]">
Array of tags associated with the flow

<ResponseField name="step_count" type="integer">
Total number of steps in the flow definition

<ResponseField name="created_at" type="string">

<ResponseField name="updated_at" type="string">

```typescript TypeScript - Pagination Helper
async function getAllFlows() {
  const allFlows = [];
  let cursor = null;
  
  do {
    const params = new URLSearchParams({ limit: '50' });
    if (cursor) params.set('cursor', cursor);
    
    const response = await fetch(`https://tolstoy.getpullse.com/flows?${params}`, {
      headers: {
        'x-org-id': 'org_1234567890abcdef',
        'x-user-id': 'user_abcdef1234567890'
      }
    });
    
    const result = await response.json();
    allFlows.push(...result.data);
    cursor = result.pagination.has_more ? result.pagination.cursor : null;
  } while (cursor);
  
  return allFlows;
}
```
