# Installation

Get started with the Tolstoy TypeScript SDK in seconds.

## Package Manager Installation

### npm
```bash
npm install @joosuhail/tolstoy-sdk
```

### Yarn
```bash
yarn add @joosuhail/tolstoy-sdk
```

### pnpm
```bash
pnpm add @joosuhail/tolstoy-sdk
```

## Package Details

- **Package Name**: `@joosuhail/tolstoy-sdk`
- **Current Version**: `1.0.0`
- **Size**: 26.8 KB compressed, 221.7 KB unpacked
- **Dependencies**: axios (production-ready)
- **TypeScript**: Full type definitions included

## Requirements

- **Node.js**: 16.0.0 or higher
- **TypeScript**: 5.0.0 or higher (for TypeScript projects)
- **Browser**: Modern browsers with ES2020 support

## Verify Installation

After installation, verify the SDK is working:

```typescript
import { TolstoyClient } from '@joosuhail/tolstoy-sdk';

console.log('Tolstoy SDK installed successfully!');
```
