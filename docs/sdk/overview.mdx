## Quick Overview

```typescript
import { TolstoyClient } from '@joosuhail/tolstoy-sdk';

const client = new TolstoyClient(
  'https://tolstoy.getpullse.com',
  'your-org-id',
  'your-user-id'
);

// OAuth2 Authentication Setup
await client.configureOAuth2({
  toolId: 'github-tool-id',
  provider: 'github',
  clientId: 'your-github-client-id',
  clientSecret: 'your-github-client-secret',
  redirectUri: 'https://your-app.com/callback'
});

// Execute workflows with automatic auth injection
const execution = await client.runFlow('flow_123', {
  userId: 'user_456',
  email: 'user@example.com'
});

// Execute standalone actions
const actionResult = await client.executeAction('github-create-issue', {
  title: 'Bug Report',
  body: 'Found an issue with the authentication flow',
  labels: ['bug', 'priority-high']
});

// Monitor system health
const health = await client.getDetailedHealth();
console.log(`System status: ${health.database.status}`);
```

## Architecture

The SDK provides two levels of access:

### 1. Helper Methods (Recommended)
Simplified methods for common operations with sensible defaults.

```typescript
// OAuth2 authentication setup
await client.setupOAuth2('github', {
  clientId: 'your-client-id',
  clientSecret: 'your-client-secret',
  redirectUri: 'https://your-app.com/callback'
});

// Simple action execution with auto-auth
const result = await client.executeAction('create-github-issue', {
  title: 'New Feature Request',
  body: 'Please add OAuth2 support'
});

// Simple workflow execution
const execution = await client.runFlow('flow_id', inputs);

// Health monitoring
const health = await client.checkHealth();
```

### 2. Raw API Access (Advanced)
Direct access to all generated endpoints for advanced use cases.

```typescript
// Advanced OAuth2 operations
const authUrl = await client.raw.auth.authControllerGetAuthorizeUrl({
  provider: 'github',
  orgId: 'org_123',
  userId: 'user_456'
});

// Direct action execution
const actionResult = await client.raw.actions.actionsControllerExecute(
  'action_id',
  { inputs: { title: 'Issue Title' } }
);

// Detailed health monitoring
const health = await client.raw.health.healthControllerGetDetailedHealth();

// Prometheus metrics access
const metrics = await client.raw.metrics.metricsControllerGetMetrics();
```

## Multi-Tenant Architecture

Built for SaaS applications with organization-level isolation:

```typescript
// Different clients for different tenants
const clientA = new TolstoyClient(baseUrl, 'org_a', 'user_123');
const clientB = new TolstoyClient(baseUrl, 'org_b', 'user_456');

// Each client operates in its own tenant context
await clientA.runFlow('flow_id', inputs); // Isolated to org_a
await clientB.runFlow('flow_id', inputs); // Isolated to org_b
```
