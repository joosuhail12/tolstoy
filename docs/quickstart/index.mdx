# Quickstart

Get Tolstoy up and running in under 5 minutes with this step-by-step guide.

## 1. Install the CLI

```bash
npm install -g @tolstoy/cli
```

## 2. Scaffold a Project

```bash
tolstoy init my-project
cd my-project
```

## 3. Define a Simple Flow

Create your first workflow with a simple hello-world example:

```yaml
version: 1
steps:
  - key: hello-world
    action: echo
    inputs:
      message: "Hello, world!"
```

## 4. Run the Flow

Execute your workflow with the Tolstoy CLI:

```bash
tolstoy run --org=org-123 --flow=my-flow
```
