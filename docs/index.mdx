# Welcome to Tolstoy

Tolstoy is a powerful workflow automation platform that enables teams to build, deploy, and manage complex workflows with ease.

## Getting Started

Ready to build your first workflow? Check out our [Quickstart Guide](/quickstart) to get started in 5 minutes.

## Key Features

- **Visual Workflow Builder** - Design complex workflows with an intuitive interface
- **Extensive Action Library** - Pre-built integrations for popular tools and services
- **Powerful API** - Full REST API for programmatic access
- **Real-time Monitoring** - Track workflow execution and performance
- **Team Collaboration** - Share and collaborate on workflows with your team
