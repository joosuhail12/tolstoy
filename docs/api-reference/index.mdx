## Authentication

All API requests require these headers for multi-tenant access:

- **x-org-id**: Your organization ID
- **x-user-id**: Your user ID for request context

## Base URLs

- **Production**: `https://tolstoy.getpullse.com`
- **Development**: `http://localhost:3000`

## Rate Limits

- **General API**: 1000 requests per minute
- **Flow Execution**: 100 requests per minute  
- **Webhooks**: 500 requests per minute
- **Metrics**: 200 requests per minute

## Quick Start Examples

### 1. Create a Tool Integration
```bash
curl -X POST https://tolstoy.getpullse.com/tools \
  -H "x-org-id: your-org-id" \
  -H "x-user-id: your-user-id" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "GitHub API",
    "baseUrl": "https://api.github.com",
    "authType": "bearer"
  }'
```

### 2. Execute a Workflow
```bash
curl -X POST https://tolstoy.getpullse.com/flows/{flowId}/execute \
  -H "x-org-id: your-org-id" \
  -H "x-user-id: your-user-id" \
  -H "Content-Type: application/json" \
  -d '{
    "mode": "async",
    "inputs": {
      "email": "user@example.com"
    }
  }'
```

### 3. Monitor Execution
```bash
curl https://tolstoy.getpullse.com/flows/{flowId}/executions/{executionId} \
  -H "x-org-id: your-org-id" \
  -H "x-user-id: your-user-id"
```

## Error Handling

All API errors follow a consistent format:

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid input parameters",
    "details": {
      "field": "name", 
      "issue": "required"
    }
  },
  "meta": {
    "timestamp": "2024-01-15T10:30:00Z",
    "requestId": "req_abc123"
  }
}
```

### Common Error Codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `VALIDATION_ERROR` | 400 | Invalid request parameters |
| `UNAUTHORIZED` | 401 | Missing or invalid authentication |
| `FORBIDDEN` | 403 | Insufficient permissions |
| `NOT_FOUND` | 404 | Resource not found |
| `CONFLICT` | 409 | Resource already exists |
| `RATE_LIMITED` | 429 | Too many requests |
| `INTERNAL_ERROR` | 500 | Server error |

## SDKs and Libraries

### JavaScript/TypeScript SDK
```bash
npm install @tolstoy/sdk
```

**Example: Execute a workflow**
```typescript
import { TolstoyClient } from '@tolstoy/sdk';

const client = new TolstoyClient({
  baseUrl: 'https://tolstoy.getpullse.com',
  headers: {
    'x-org-id': 'your-org-id',
    'x-user-id': 'your-user-id'
  }
});

// Execute workflow with type safety
const execution = await client.flows.execute('flow_abc123', {
  variables: {
    userId: 'user_123',
    email: 'user@example.com'
  },
  useDurable: true
});

console.log(`Execution started: ${execution.executionId}`);
```

### Python SDK
```bash
pip install tolstoy-python
```

**Example: Monitor workflow execution**
```python
import tolstoy
from tolstoy import TolstoyClient

client = TolstoyClient(
    base_url="https://tolstoy.getpullse.com",
    headers={
        "x-org-id": "your-org-id",
        "x-user-id": "your-user-id"
    }
)

# Execute and monitor workflow
execution = client.flows.execute(
    flow_id="flow_abc123",
    variables={"userId": "user_123"},
    use_durable=True
)

# Poll for completion
status = client.flows.get_execution_status(
    flow_id="flow_abc123", 
    execution_id=execution.execution_id
)
print(f"Status: {status.status}")
```

### Go SDK
```bash
go get github.com/tolstoy/go-sdk
```

**Example: Workflow automation**
```go
package main

import (
    "context"
    "fmt"
    "github.com/tolstoy/go-sdk"
)

func main() {
    client := tolstoy.NewClient(&tolstoy.Config{
        BaseURL: "https://tolstoy.getpullse.com",
        Headers: map[string]string{
            "x-org-id":  "your-org-id",
            "x-user-id": "your-user-id",
        },
    })

    // Execute workflow
    execution, err := client.Flows.Execute(context.Background(), &tolstoy.ExecuteFlowRequest{
        FlowID: "flow_abc123",
        Variables: map[string]interface{}{
            "userId": "user_123",
            "email":  "user@example.com",
        },
        UseDurable: true,
    })
    
    if err != nil {
        panic(err)
    }
    
    fmt.Printf("Execution ID: %s\n", execution.ExecutionID)
}
