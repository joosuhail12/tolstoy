# List Executions for Specific Action

Get execution history for a specific action

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="key" type="string" required>
  Action key
</ParamField>

<ParamField header="X-Status" type="string" required>
  
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:3000/actions/{key}/executions" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc"
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.actions.actionscontroller_listactionexecutions();
console.log(result);
```
</CodeGroup>

## Responses

### 200 OK

Successfully retrieved action executions
