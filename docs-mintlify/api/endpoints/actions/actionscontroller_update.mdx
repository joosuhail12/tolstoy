# Update Action

Update an action template configuration

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="id" type="string" required>
  Action ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X PUT "http://localhost:3000/actions/{id}" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.actions.actionscontroller_update();
console.log(result);
```
</CodeGroup>

## Responses

### 200 OK

Action updated successfully

### 400 Bad Request

Invalid action configuration

### 404 Not Found

Action not found
