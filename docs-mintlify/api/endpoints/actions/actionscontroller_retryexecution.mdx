# Retry Execution

Retry a failed or cancelled action execution

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="executionId" type="string" required>
  Execution ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:3000/actions/executions/{executionId}/retry" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.actions.actionscontroller_retryexecution();
console.log(result);
```
</CodeGroup>

## Responses

<ResponseExample>
```json 202 
{
  "success": true,
  "executionId": "exec_9876543210_xyz789",
  "duration": 1250,
  "data": {},
  "outputs": {}
}
```
</ResponseExample>

### 400 Bad Request

Can only retry failed or cancelled executions

### 404 Not Found

Execution not found
