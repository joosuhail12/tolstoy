# Execute a single Action by ID

Execute a standalone action with provided inputs. This endpoint allows you to run individual actions outside of workflow contexts.

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField header="X-Org-ID" type="string" required>
  Organization ID
</ParamField>

<ParamField header="X-User-ID" type="string">
  User ID for user-scoped authentication (optional)
</ParamField>

<ParamField path="id" type="string" required>
  Action ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:3000/actions/{id}/execute" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.actions.actionscontroller_execute();
console.log(result);
```
</CodeGroup>
