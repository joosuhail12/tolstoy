# List Webhooks

Get all webhooks for the organization, optionally filtered by event type

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField query="eventType" type="string">
  Filter webhooks by event type
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:3000/webhooks" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc"
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.webhooks.webhookscontroller_findall();
console.log(result);
```
</CodeGroup>
