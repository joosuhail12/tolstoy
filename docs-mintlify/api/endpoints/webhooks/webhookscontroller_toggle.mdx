# Toggle Webhook

Enable or disable a webhook

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="id" type="string" required>
  Webhook ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X PATCH "http://localhost:3000/webhooks/{id}/toggle" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.webhooks.webhookscontroller_toggle();
console.log(result);
```
</CodeGroup>

## Responses

<ResponseExample>
```json 200 OK
{
  "id": "webhook_abc123",
  "enabled": false,
  "updatedAt": "2024-01-15T10:30:00.000Z"
}
```
</ResponseExample>

### 404 Not Found

Webhook not found
