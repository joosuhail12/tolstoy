# Version Information

Get version information including build details and commit hash

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>





## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:3000/version" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc"
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.version.appcontroller_getversion();
console.log(result);
```
</CodeGroup>

## Responses

<ResponseExample>
```json 200 OK
{
  "version": "1.0.0",
  "commit": "abc123def456",
  "buildTime": "2025-08-09T10:30:00.000Z",
  "nodeVersion": "20.11.0",
  "environment": "production"
}
```
</ResponseExample>
