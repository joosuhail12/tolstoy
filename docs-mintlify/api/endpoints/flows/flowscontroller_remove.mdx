# Delete Flow

Delete a workflow permanently

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="id" type="string" required>
  Flow ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X DELETE "http://localhost:3000/flows/{id}" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.flows.flowscontroller_remove();
console.log(result);
```
</CodeGroup>

## Responses

### 204 No Content

Flow deleted successfully

### 404 Not Found

Flow not found

### 409 Conflict

Flow has active executions and cannot be deleted
