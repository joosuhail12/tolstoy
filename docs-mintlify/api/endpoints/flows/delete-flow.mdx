---
title: "Delete Flow"
api: "DELETE /flows/{id}"
description: "Delete a flow and optionally its associated execution history"
---

## Overview

The Delete Flow endpoint permanently removes a flow from your organization. You can optionally preserve the execution history for audit purposes.

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the flow to delete
</ParamField>

## Query Parameters

<ParamField query="preserveHistory" type="boolean" default="false">
  If true, execution history will be preserved after deletion
</ParamField>

<ParamField query="force" type="boolean" default="false">
  Force deletion even if there are active executions
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="deletedAt" type="string">
  ISO 8601 timestamp when the flow was deleted
</ResponseField>

<ResponseField name="historyPreserved" type="boolean">
  Whether execution history was preserved
</ResponseField>

<ResponseField name="executionsArchived" type="number">
  Number of executions archived (if history preserved)
</ResponseField>

<ResponseExample>
```json
{
  "success": true,
  "message": "Flow successfully deleted",
  "deletedAt": "2024-01-15T11:30:00Z",
  "historyPreserved": true,
  "executionsArchived": 156
}
```
</ResponseExample>

## Error Responses

### 404 Not Found
Flow with the specified ID does not exist.

```json
{
  "error": {
    "code": "FLOW_NOT_FOUND",
    "message": "Flow with ID flow_abc123 not found"
  }
}
```

### 409 Conflict
Flow has active executions and force flag is not set.

```json
{
  "error": {
    "code": "ACTIVE_EXECUTIONS",
    "message": "Flow has 3 active executions. Use force=true to delete anyway"
  }
}
```

### 403 Forbidden
Insufficient permissions to delete the flow.

```json
{
  "error": {
    "code": "INSUFFICIENT_PERMISSIONS",
    "message": "You do not have permission to delete this flow"
  }
}
```
