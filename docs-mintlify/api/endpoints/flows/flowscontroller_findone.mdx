# Get Flow

Get a specific workflow by ID

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="id" type="string" required>
  Flow ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:3000/flows/{id}" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc"
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.flows.flowscontroller_findone();
console.log(result);
```
</CodeGroup>

## Responses

### 200 OK

Successfully retrieved flow

### 404 Not Found

Flow not found
