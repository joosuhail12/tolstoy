# Execute Flow

Execute a workflow either synchronously or asynchronously with optional input variables

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="id" type="string" required>
  Flow ID to execute
</ParamField>

<ParamField body="variables" type="object">
  Input variables for the flow execution
</ParamField>

<ParamField body="useDurable" type="boolean">
  Whether to use durable (async) execution
</ParamField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:3000/flows/{id}/execute" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.flows.flowscontroller_executeflow();
console.log(result);
```
</CodeGroup>

## Responses

<ResponseExample>
```json 202 
{
  "executionId": "exec_abc123",
  "status": "running",
  "mode": "durable"
}
```
</ResponseExample>

### 400 Bad Request

Invalid execution parameters

### 404 Not Found

Flow not found
