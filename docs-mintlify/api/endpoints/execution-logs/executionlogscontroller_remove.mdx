# Delete Execution Log

Delete an execution log entry. Use with caution as this removes audit trail information.

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="id" type="string" required>
  Execution log ID
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X DELETE "http://localhost:3000/execution-logs/{id}" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.execution-logs.executionlogscontroller_remove();
console.log(result);
```
</CodeGroup>

## Responses

### 204 No Content

Execution log deleted successfully

### 404 Not Found

Execution log not found

### 409 Conflict

Cannot delete log entry that is part of active execution
