# List Execution Logs

Get all execution logs for the organization. This provides a complete history of workflow step executions.

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>





## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:3000/execution-logs" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc"
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.execution-logs.executionlogscontroller_findall();
console.log(result);
```
</CodeGroup>
