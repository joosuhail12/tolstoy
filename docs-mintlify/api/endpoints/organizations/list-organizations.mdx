## Overview

The List Organizations endpoint returns all organizations that the authenticated user has access to, along with their basic information, usage statistics, and user role within each organization.

## Authentication

<ParamField header="authorization" type="string" required>
  Bearer token for API authentication
</ParamField>

## Query Parameters

<ParamField query="limit" type="number">
  Maximum number of organizations to return (1-100, defaults to 20)
</ParamField>

<ParamField query="offset" type="number">
  Number of organizations to skip for pagination (defaults to 0)
</ParamField>

<ParamField query="sort" type="string">
  Sort order: "name", "created", "updated", or "usage" (defaults to "name")
</ParamField>

<ParamField query="order" type="string">
  Sort direction: "asc" or "desc" (defaults to "asc")
</ParamField>

<ParamField query="role" type="string">
  Filter by user role: "owner", "admin", "member", or "viewer"
</ParamField>

<ParamField query="plan" type="string">
  Filter by billing plan: "free", "starter", "professional", or "enterprise"
</ParamField>

<ParamField query="search" type="string">
  Search organizations by name or description
</ParamField>

## Usage Examples

<CodeGroup>

```bash cURL Basic
curl -X GET "https://api.tolstoy.com/v1/organizations" \
  -H "Authorization: Bearer $API_TOKEN"
```

```bash cURL Filtered
curl -X GET "https://api.tolstoy.com/v1/organizations?role=owner&plan=professional&limit=10&sort=usage&order=desc" \
  -H "Authorization: Bearer $API_TOKEN"
```

```javascript JavaScript
// Basic list
const organizations = await fetch('/api/v1/organizations', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${apiToken}`
  }
});

const orgData = await organizations.json();
console.log(`Found ${orgData.pagination.total} organizations`);

// Filtered list with pagination
const filteredOrgs = await fetch('/api/v1/organizations?role=owner&plan=professional&limit=10', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${apiToken}`
  }
});

const filteredData = await filteredOrgs.json();
filteredData.organizations.forEach(org => {
  console.log(`${org.name}: ${org.usage.workflows} workflows, ${org.usage.users} users`);
});
```

```python Python
import requests

# Basic list
response = requests.get(
    "https://api.tolstoy.com/v1/organizations",
    headers={
        "Authorization": f"Bearer {api_token}"
    }
)

if response.status_code == 200:
    data = response.json()
    print(f"Found {data['pagination']['total']} organizations")
    
    for org in data['organizations']:
        print(f"- {org['name']} ({org['plan']} plan, {org['userRole']} role)")

# Filtered and paginated list
filtered_response = requests.get(
    "https://api.tolstoy.com/v1/organizations",
    headers={
        "Authorization": f"Bearer {api_token}"
    },
    params={
        "role": "owner",
        "plan": "professional",
        "limit": 10,
        "sort": "usage",
        "order": "desc"
    }
)

if filtered_response.status_code == 200:
    filtered_data = filtered_response.json()
    for org in filtered_data['organizations']:
        usage = org['usage']
        print(f"{org['name']}: {usage['workflows']} workflows, {usage['users']} users")
```

## Filtering and Sorting

### Available Filters
- **role**: Filter by your role in the organization
- **plan**: Filter by billing plan
- **search**: Text search in name and description
- **status**: Filter by organization status

### Sort Options
- **name**: Alphabetical by organization name
- **created**: By creation date
- **updated**: By last update time
- **usage**: By current usage metrics (workflows + executions)

## Pagination

Use `limit` and `offset` parameters for pagination:
- Maximum `limit` is 100
- Use `hasMore` field to determine if more results exist
- Calculate next offset: `current_offset + limit`

## Response Optimization

The endpoint automatically optimizes responses based on your access level:
- **Owners/Admins**: See full usage statistics and billing information
- **Members/Viewers**: See limited usage statistics
- **Suspended orgs**: Only basic information is returned
