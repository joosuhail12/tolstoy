# Delete Tool Credentials

Permanently delete stored credentials for a tool. This will affect any workflows that use this tool.

## Authentication

<ParamField header="x-org-id" type="string" required>
  Your organization ID
</ParamField>

<ParamField header="x-user-id" type="string" required>
  Your user ID
</ParamField>

<ParamField path="toolId" type="string" required>
  Tool ID to delete credentials for
</ParamField>



## Examples

<CodeGroup>
```bash cURL
curl -X DELETE "http://localhost:3000/tools/{toolId}/secrets" \
  -H "Content-Type: application/json" \
  -H "x-org-id: org_abc123def456" \
  -H "x-user-id: user_xyz789abc" \
  -d '{}'
```

```javascript JavaScript SDK
import { TolstoyClient } from '@tolstoy/sdk';

const tolstoy = new TolstoyClient({
  orgId: 'org_abc123def456',
  userId: 'user_xyz789abc'
});

const result = await tolstoy.tools.toolsecretscontroller_deletecredentials();
console.log(result);
```
</CodeGroup>

## Responses

### 204 No Content

Credentials deleted successfully

### 404 Not Found

Tool credentials not found

### 409 Conflict

Cannot delete credentials that are being used by active workflows
