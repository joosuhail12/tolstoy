## Overview

The List Executions endpoint retrieves all workflow executions within your organization, with filtering and pagination options to help you find specific executions.

## Authentication

<ParamField header="x-org-id" type="string" required>
  Organization ID for multi-tenant access control
</ParamField>

<ParamField header="x-user-id" type="string" required>
  User ID of the requesting user for authorization
</ParamField>

<ParamField header="authorization" type="string" required>
  Bearer token for API authentication
</ParamField>

## Query Parameters

<ParamField query="limit" type="number">
  Maximum number of executions to return (1-100, defaults to 20)
</ParamField>

<ParamField query="offset" type="number">
  Number of executions to skip for pagination (defaults to 0)
</ParamField>

<ParamField query="workflowId" type="string">
  Filter by specific workflow ID
</ParamField>

<ParamField query="status" type="string">
  Filter by execution status: "pending", "running", "completed", "failed", "canceled"
</ParamField>

<ParamField query="triggeredBy" type="string">
  Filter by user who triggered the execution
</ParamField>

<ParamField query="startedAfter" type="string">
  Filter executions started after this ISO timestamp
</ParamField>

<ParamField query="startedBefore" type="string">
  Filter executions started before this ISO timestamp
</ParamField>

<ParamField query="sort" type="string">
  Sort order: "created", "updated", "duration", or "status" (defaults to "created")
</ParamField>

<ParamField query="order" type="string">
  Sort direction: "asc" or "desc" (defaults to "desc")
</ParamField>

```javascript JavaScript
  method: 'GET',

    'Authorization': `Bearer ${apiToken}`,
    'X-Org-ID': 'org_123456',
    'X-User-ID': 'user_789012'

const execData = await executions.json();
console.log(`Found ${execData.summary.totalExecutions} total executions`);

// Filtered list
const runningExecutions = await fetch('/api/v1/executions?status=running&sort=created&order=desc', {

  headers: {
    'Authorization': `Bearer ${apiToken}`,
    'X-Org-ID': 'org_123456',
    'X-User-ID': 'user_789012'
  }
});

const runningData = await runningExecutions.json();
runningData.executions.forEach(exec => {
  console.log(`${exec.workflowName}: ${exec.progress}% complete`);
});
```

## Filtering Options

### By Status
- `pending` - Executions waiting to start
- `running` - Currently executing workflows
- `completed` - Successfully finished executions
- `failed` - Executions that encountered errors
- `canceled` - Manually canceled executions

### By Time Range
```javascript
// Last 24 hours
const yesterday = new Date(Date.now() - 24 * 60 * 60 * 1000).toISOString();
const recentExecs = await fetch(`/api/v1/executions?startedAfter=${yesterday}`);

// Specific date range
const startDate = '2024-01-01T00:00:00Z';
const endDate = '2024-01-31T23:59:59Z';
const monthlyExecs = await fetch(`/api/v1/executions?startedAfter=${startDate}&startedBefore=${endDate}`);
```

### By Workflow
```javascript
// All executions for specific workflow
const workflowExecs = await fetch('/api/v1/executions?workflowId=workflow_123');

// Failed executions for specific workflow
const failedExecs = await fetch('/api/v1/executions?workflowId=workflow_123&status=failed');
```

## Use Cases

### Monitoring Dashboard
```javascript
// Get current system status
const status = await fetch('/api/v1/executions?limit=1');
const statusData = await status.json();
const summary = statusData.summary;

console.log(`System Status:
Running: ${summary.runningExecutions}
Avg Duration: ${Math.round(summary.averageDuration/1000)}s
Success Rate: ${((summary.completedExecutions / summary.totalExecutions) * 100).toFixed(1)}%`);
```

### Execution History
```javascript
// Get execution history for a workflow
const history = await fetch('/api/v1/executions?workflowId=workflow_123&sort=created&order=desc&limit=20');
const historyData = await history.json();

historyData.executions.forEach(exec => {
  const duration = exec.duration ? `${Math.round(exec.duration/1000)}s` : 'Running';
  console.log(`${exec.startedAt}: ${exec.status} (${duration})`);
});
```

### Troubleshooting
```javascript
// Find recent failures
const failures = await fetch('/api/v1/executions?status=failed&sort=created&order=desc&limit=10');
const failureData = await failures.json();

failureData.executions.forEach(exec => {
  console.log(`Failed: ${exec.workflowName} at ${exec.startedAt}`);
});
```
