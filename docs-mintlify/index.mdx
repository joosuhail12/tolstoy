Tolstoy is a powerful workflow automation platform that enables teams to build, deploy, and manage complex automation workflows with ease. Whether you're automating business processes, integrating systems, or building custom workflows, Tolstoy provides the tools and infrastructure you need to succeed.

## Example Workflow

Here's a simple example of creating a workflow that sends a Slack notification when a new user signs up:

```javascript
const workflow = {
  name: "User Signup Notification",
  trigger: {
    type: "webhook",
    path: "/user-signup"
  },
  actions: [
    {
      type: "slack",
      channel: "#general",
      message: "New user signed up: {{trigger.body.email}}"
    }
  ]
};
```
